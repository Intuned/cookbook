# Templates

Complete, ready-to-deploy Intuned projects that you can use as starting points for your own browser automations.

## Available Templates

| Template | Description |
|----------|-------------|
| [book-consultations](./book-consultations/) | Booking automation to book consultations and list bookings (no authentication) |
| [book-consultations-auth](./book-consultations-auth/) | Authenticated booking automation with session management using Intuned Auth Sessions |
