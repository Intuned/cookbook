# Book Consultations with Auth

Credentials-based booking automation to book a consultation with a consultant and list the consultations.

This template demonstrates how to build authenticated browser automations using Intuned Auth Sessions. It handles user login, session management, and provides APIs for booking consultations and retrieving existing bookings by email.

## Features

- **Authenticated Sessions**: Uses Intuned Auth Sessions for secure, credential-based automation
- **Book Consultations**: Automate the booking of consultations with consultants
- **List Consultations**: Retrieve consultations by email address
